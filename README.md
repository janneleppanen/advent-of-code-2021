# Advent of Code 2021

## Running a task

Solve all puzzles:  
`npm start`

Solve a single puzzle:  
`npm start 1`

## Running a test

Run tests for all puzzles:  
`npm run test`

Run tests for a single puzzle:  
`npm run test day-1`
